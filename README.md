# 战棋推演沙盘

一个基于 Canvas 的战棋推演工具，支持布局阶段和推演阶段，可绘制地形、放置棋子、记录移动路径。

## 项目结构

```
wargame-sandbox/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── app.js          # 应用逻辑
└── README.md           # 说明文档
```

## 功能特性

### 🎮 双阶段模式
- **布局阶段**: 放置棋子、绘制地形、设置棋子属性
- **推演阶段**: 拖拽移动棋子，自动寻路或手绘路径

### ⛰️ 地形系统
- 水（💧）: 需要跳跃能力才能通过
- 墙（🧱）: 不可通行
- 陷阱（⚠️）: 特殊地形
- 箱子（📦）: 障碍物
- 自定义地形: 支持添加自定义标签地形

### 🎲 棋子系统
- 红蓝双方棋子
- 跳跃能力: 可无视水域
- 自动编号回收系统

### 📜 记录系统
- 自动记录每一步移动
- 支持添加备注
- 回退到任意步骤
- 查看历史状态
- 保存/加载棋盘状态

### 📐 棋盘调整
- 自定义棋盘大小 (3x3 ~ 30x30)
- 扩大/缩小棋盘（保持内容居中）

## 使用说明

1. **布局阶段**:
   - 点击红/蓝棋子仓库选中棋子类型
   - 点击棋盘放置棋子
   - 点击棋子选中，可勾选"跳跃"属性
   - 拖拽棋子移动位置
   - 选择地形工具，点击/拖拽绘制地形

2. **推演阶段**:
   - 长按棋子拖拽到目标位置
   - 自动寻路模式: 系统自动计算路径
   - 手绘路径模式: 手动绘制移动路径
   - 移动记录会自动保存

3. **保存/加载**:
   - 点击"保存棋盘"将当前状态保存为 JSON 文件
   - 点击"读取棋盘"加载之前保存的状态

## 文件说明

### index.html
主页面结构，包含:
- 棋盘区域（Canvas）
- 控制面板（棋子仓库、地形工具、阶段切换）
- 移动记录区域

### css/style.css
样式文件，包含:
- 整体布局和响应式设计
- 棋盘和控制面板样式
- 按钮和交互元素样式
- 移动端适配

### js/app.js
应用逻辑，主要模块:
- **数据管理**: 棋盘、棋子、地形、路径等数据
- **绘制系统**: Canvas 渲染棋盘、棋子、路径
- **事件处理**: 鼠标/触摸事件、拖拽逻辑
- **移动系统**: 布局移动、推演移动（自动寻路/手绘）
- **记录系统**: 移动日志、快照保存/恢复
- **文件操作**: 保存/加载 JSON 文件

## 开发说明

### 修改样式
编辑 `css/style.css` 文件，主要样式类:
- `.board-area`: 棋盘区域
- `.right-panel`: 右侧面板
- `.piece`: 棋子样式
- `.terrain-btn`: 地形按钮

### 修改逻辑
编辑 `js/app.js` 文件，主要函数:
- `drawBoard()`: 绘制棋盘
- `placePiece()`: 放置棋子
- `setTerrain()`: 设置地形
- `playMovePieceAuto()`: 自动寻路移动
- `playMovePieceFreehand()`: 手绘路径移动
- `saveBoardToFile()`: 保存棋盘
- `loadBoardFromFile()`: 加载棋盘
