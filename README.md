
## 功能特性

### 🎮 双阶段模式
- **布局阶段**：放置棋子、绘制地形、设置棋子属性
- **推演阶段**：拖拽移动棋子、调速、击退、换位、手绘路径

### ⛰️ 地形系统
- 水（💧）：需要跳跃能力才能通过
- 墙（🧱）：不可通行
- 陷阱（⚠️）：特殊地形
- 箱子（📦）：障碍物
- 自定义地形：支持添加自定义标签地形（可自定义颜色）

### 🎲 棋子系统
- 己方/敌方/自定义角色
- 速度值决定行动顺序
- 调速 Buff：临时增加或减少速度，可持续指定行动次数
- 支持 3×3 大型单位（敌方）
- 行动顺序实时显示在棋子上

### ⚔️ 推演操作
- **移动**：拖拽棋子到目标格子，自动寻路或手绘路径
- **行动结束**：手动结束当前棋子回合，切换至下一个棋子
- **调速**：为任意棋子添加临时速度变化（可指定次数）
- **击退**：选择目标棋子，拖拽方向将其沿直线或斜线击退
- **换位**：依次点击两个棋子，交换位置（禁止与大型单位换位）
- **清除路径**：删除所有移动路径与幽灵标记
- **重置推演**：恢复到进入推演时的初始状态

### 📜 记录系统
- 自动记录每一步移动、调速、击退、换位
- 支持添加备注
- 回退至任意步骤
- 查看历史状态
- 保存/加载完整棋盘状态（含移动记录）

### 📐 棋盘调整
- 自定义棋盘大小 (3x3 ~ 30x30)
- 扩大/缩小棋盘（保持内容居中）

## 使用说明

### 布局阶段
1. 点击“己方/自定义/敌方”按钮选择棋子类型
2. 点击棋盘格子放置棋子
3. 拖拽棋子可调整位置
4. 选择地形工具（水/墙/陷阱/箱子/自定义），点击或拖拽绘制地形

### 推演阶段
1. **移动棋子**：长按当前行动棋子并拖拽到目标格子
   - 关闭手绘模式时：系统自动计算最短路径
   - 开启手绘模式时：可自由绘制移动路径
2. **调速**：点击“调速”按钮，选择速度变化值与生效次数，然后点击目标棋子
3. **击退**：点击“击退”按钮，再点击要击退的棋子，然后向外拖拽指定方向（直线或斜线）
4. **换位**：点击“换位”按钮，依次点击两个棋子交换位置
5. **行动结束**：点击“行动结束”按钮，当前棋子行动结束，轮到下一个棋子
6. **重置推演**：恢复到进入推演时的布局状态（所有推演步骤清空）
7. **保存/加载**：保存当前棋盘（含推演记录）为 JSON 文件，或加载之前保存的棋盘

### 其他功能
- **移动记录**：可添加备注，回退或查看历史步骤
- **地形标记**：推演模式下可放置陷阱/箱子/旗子/火/冰等临时标记，不影响移动
- **自定义地形/标记**：布局模式可自定义地形名称与颜色；推演模式可自定义标记

## 开发说明

### 样式修改
编辑 `css/style.css`，主要类：
- `.board-area`：棋盘区域
- `.right-panel`：右侧面板
- `.terrain-btn`：地形/标记按钮
- `.play-action-grid`：推演操作按钮网格

### 逻辑修改
核心逻辑分布在以下 JS 文件中：
- `state.js`：全局状态（棋子、地形、移动记录、快照等）
- `game.js`：移动、调速、击退、换位、保存/加载等
- `movement.js`：寻路、行动顺序、通行判断
- `input.js`：鼠标/触摸事件处理（含击退拖拽、换位点击）
- `ui.js`：界面交互、模式切换、日志显示
- `renderer.js`：Canvas 绘制（棋子、地形、路径、高亮、提示）
